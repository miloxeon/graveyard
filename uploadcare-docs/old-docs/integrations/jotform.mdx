# Serverless File Uploads for JotForm

lots of convenient integrations. The Uploadcare [integration][ext-jotform-uploader]

The File Uploader is a highly customizable file uploader that adapts to your flows and
layouts.

By the way, every file uploaded via Uploadcare gets instantly cached on our CDN
and swiftly and reliably delivered across 130+ countries.

In this guide, we are going to create a form that collects speaker candidate
data before some conference. We will use a combination of JotForm fields and two
file uploaders. The first one deals with uploading a candidate photo and making some
edits, and the second one will be up to collecting slides for a presentation.
