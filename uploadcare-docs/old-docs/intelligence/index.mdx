# Intelligence

Uploadcare provides you with powerful and intelligent computer vision features
for image analysis. It helps you find certain objects, faces, and colors for
further processing and image classification. Also, you can use Intelligence for
file validation upon uploading, generating image alt texts, etc.

Access [Intelligence][intelligence-product-page] via:

* Instant CDN operations such as [Face Detection][intelligence-face-detection], 
  [Smart Compression][transformations-image-smart-compression], etc.
* REST API for [Object Recognition][intelligence-object-recognition]
  (experimental).

Smart Features like [Smart Crop][transformations-image-smart-crop] and [Blur Faces][transformations-blur-region-faces]
use the Intelligence core as well. It'll select areas of an image and then it'll
automatically apply a certain [Image Transformation][transformations-image-list].
